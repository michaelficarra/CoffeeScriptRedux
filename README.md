If you haven't seen [the funded Kickstarter project](http://www.kickstarter.com/projects/michaelficarra/make-a-better-coffeescript-compiler)
yet, you should check that out to find out more about this.

### Getting Started

    npm install
    make -j test
    bin/coffee --help

### Development

    make -j build test
